「プロの力が身につく iPhone/iPadアプリケーション開発の教科書 Swift対応版」サンプルコード集
===========
ダウンロードしてファイルを解凍すると以下の二つのフォルダに分かれています。

* ios
* ios_swift

ios_swiftフォルダにこの本のサンプルコードが入っています(iosフォルダには初版のコードが入っています)。

サンプルコードの概要
